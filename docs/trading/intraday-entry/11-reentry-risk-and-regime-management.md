# 第十一章：重新入场、每日风险与环境适应

> 止损后的下一笔交易必须重新建立理由；每日风险预算必须限制连续错误；持仓节奏必须适应当天的市场环境。

## 本章总流程

```text
第一次交易结束
→ 判断是正常止损还是结构失效
→ 等待新的位置与新的①②③
→ 检查剩余风险预算
→ 根据趋势、震荡或反转环境调整仓位与目标
→ 交易结束后记录实际R与执行偏差
```

## 一、止损后的重新入场

第一次止损结束的是旧计划，不一定代表当天没有下一次机会。但下一次交易必须当成一笔完全独立的新交易。

<iframe src="./assets/reentry-after-stop.html" title="止损后的重新入场" loading="lazy" style="width:100%;height:500px;border:0;display:block;"></iframe>

### 1. 可以重新评估的条件

重新入场前，需要重新确认：

- 价格是否回到新的需求区、供给区或关键节点；
- 5分钟环境是否仍然支持原方向；
- 是否重新出现①突破；
- 是否出现②回踩守住；
- 是否出现新的③确认；
- 新止损是否能够明确放在结构外；
- 新目标空间是否足够；
- 当日剩余风险是否允许开仓。

这不是“把第一次交易买回来”，而是建立第二笔有独立逻辑的交易。

### 2. 不能立即追回的情况

以下情况不属于重新入场：

- 止损后马上出现一根大阳线就追入；
- 价格快速收回，但没有回踩确认；
- 因为不甘心亏损而想赚回来；
- 新入场位置已经接近目标或供给区；
- 价格仍然在原来的震荡区间内，没有新的结构；
- 当日剩余风险不足以承担完整计划。

### 3. 重新入场的风险计算

假设第一次交易亏损 `-1R`，而每日上限是 `2R`，理论上最多只剩 `1R` 的风险预算。

```text
第一次交易：-1R
剩余预算：1R
下一笔计划风险：必须 ≤ 1R
第二笔再次止损：达到 -2R，停止交易
```

如果当天已经出现两笔完整止损，就不能因为出现一个看起来漂亮的新③而继续开仓。

## 二、每日风险预算与交易熔断

每日风险预算不是为了预测今天会不会赚钱，而是为了防止执行质量下降后继续扩大损失。

<iframe src="./assets/daily-risk-budget.html" title="每日风险预算与交易熔断" loading="lazy" style="width:100%;height:500px;border:0;display:block;"></iframe>

### 1. 训练阶段的风险模板

下面只是一套用于学习的示范模板，不是固定标准：

```text
单笔风险：1R
每日最大风险：2R
连续两笔完整止损：停止主动交易
达到每日上限：停止主动交易
出现规则违规：下一笔风险减半或停止
```

个人风险比例应根据账户规模、产品点值、期权最小合约和实际滑点单独制定。

### 2. 剩余风险不足怎么办？

如果每日上限是 `2R`，前面已经亏损 `1.5R`，后面即使出现机会，也不能继续按完整 `1R` 开仓。

可以选择：

1. 只做风险更小的观察仓；
2. 等待更高质量、止损更紧凑的结构；
3. 直接停止交易。

不能为了凑出完整仓位而缩短结构止损，也不能用亏损补仓来“降低平均成本”。

### 3. 相关仓位必须合并计算

SPX、ES、NQ、QQQ以及高度相关的科技股，方向一致时不能简单视为互不相关的交易。

```text
总风险 = SPX风险 + ES风险 + 相关股票风险
```

实际计算时还要考虑相关性、同时触发止损的可能性以及期权价差扩大造成的额外滑点。

### 4. 必须停止的情况

- 当日达到风险上限；
- 连续两笔完整止损；
- 开始为了翻本加大仓位；
- 连续追价或亏损补仓；
- 数据延迟、报价异常或执行系统不稳定；
- 情绪已经明显影响下单和退出。

停止交易不是失败，而是执行风险管理的一部分。

## 三、不同环境下的持仓管理

同样的③，在趋势日、震荡日和快速反转日，管理方式不能完全相同。

<iframe src="./assets/regime-management.html" title="三种市场环境下的持仓管理" loading="lazy" style="width:100%;height:520px;border:0;display:block;"></iframe>

### 1. 趋势环境：让利润运行

常见特征：

- 5分钟持续形成更高高点和更高低点，或更低低点和更低高点；
- 价格主要运行在VWAP一侧；
- 回踩后能够快速重新启动；
- 跨市场大体同步；
- 前方仍有足够目标空间。

管理方式：

```text
③确认 → 建仓
产生位移 → 持有
形成新高低点 → 上移止损
第一目标 → 分批止盈
剩余仓位 → 让趋势运行
```

趋势日最大的错误是太早全平，或者刚有一点浮盈就把止损推到成本价。

### 2. 震荡环境：快进快出

常见特征：

- 价格反复穿越VWAP；
- 上下影线较多；
- 突破后很快回到原区间；
- 高低点没有持续扩展；
- 价格在两个节点之间来回运行。

管理方式：

- 仓位折扣；
- 不在区间中间建仓；
- 靠近边界才考虑交易；
- 目标设置得更近；
- 入场后没有位移就快速减仓或退出；
- 不因为价格更便宜而加仓。

震荡日不是完全不能交易，而是不能使用趋势日的持仓期待。

### 3. 快速反转环境：先保护本金

常见特征：

- 原趋势被大实体K线破坏；
- 价格快速穿越VWAP或关键边界；
- 反弹和回踩速度明显加快；
- 目标距离很近；
- 供给区或需求区就在入场附近。

管理方式：

```text
仓位折扣
→ 目标前置
→ 快速兑现
→ 不追末端冲刺
→ 结构反向立即退出
```

即使出现漂亮的③，也不能按趋势日满仓处理。

## 四、数据的优先级

### 1. 价格位置与结构：主证据

先判断价格在哪里，是否完成①②③，止损放在哪里，目标空间是否足够。

### 2. 5分钟环境：方向过滤

5分钟负责判断今天更像趋势、震荡还是反转，不能让一根1分钟K线完全推翻更高周期环境。

### 3. VWAP与跨市场同步：确认质量

- 稳定在VWAP上方：多头环境更有利；
- 稳定在VWAP下方：空头环境更有利；
- 反复穿越VWAP：警惕震荡；
- SPX与ES/NQ不同步：仓位折扣或等待。

### 4. GEX、资金流与节点：目标和折扣

它们更适合辅助：

- 判断目标区；
- 判断哪里可能减仓；
- 判断是否需要降低仓位；
- 判断价格是否接近潜在阻力。

它们不能替代价格结构的确认。

### 5. 期权权利金：执行层数据

期权权利金主要用于评估：

- 买卖价差；
- 计划止损下的金额风险；
- 时间价值损耗；
- 是否需要分批止盈。

不能因为权利金突然上涨，就把它当成新的建仓信号；也不能因为权利金下跌，就自动认为SPX结构已经失效。

## 本章总结

```text
止损结束旧计划，不结束所有机会。
重新入场必须有新的位置和新的①②③。
风险预算减少后，仓位也必须减少。
趋势环境让利润运行。
震荡环境快进快出。
快速反转环境先保护本金。
价格结构是主证据，其他数据负责过滤和调整。
```
