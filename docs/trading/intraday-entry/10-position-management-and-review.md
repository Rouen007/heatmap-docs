# 第十章：持仓管理与交易复盘

> 建仓只是交易的一半。入场后要根据价格是否产生位移、是否形成新结构以及是否接近目标节点，决定继续持有、减仓、加仓或退出。

## 本章总流程

```text
③确认建仓
→ 观察是否产生位移
→ 第一目标分批止盈
→ 新结构形成后上移止损
→ 只在盈利的新结构后加仓
→ 接近供给区继续兑现
→ 结构失效或时间失效时退出
→ 记录计划、执行和结果
```

## 一、入场后先看价格行为，不先看盈亏

1分钟用于执行和管理，5分钟用于重新检查环境。入场后重点观察价格有没有按照预期离开建仓区。

### 1. 正常持仓：产生位移

如果价格向预期方向移动，并形成新的高点和更高低点，说明交易正在发展，可以继续持有。

### 2. 时间失效：价格不动

学习阶段可以观察入场后约2—4根1分钟K线。如果价格仍然围绕入场位横盘，说明预期位移没有出现，可以减仓或退出。

时间失效不是固定闹钟，而是“市场在应该推进的位置没有推进”。

### 3. 结构失效：主动退出

如果价格实体收盘跌破关键结构，反抽也无法重新站回，就不需要等到最远硬止损才退出。

```text
影线测试、收盘收回 → 观察
收盘跌破、反抽失败 → 退出
止损真实成交 → 接受退出，等待新的③
```

## 二、第一目标先兑现，剩余仓位跟随

目标不是一个必须精确打到的数字，而是前高、供给区、节点或其他重要边界形成的目标区域。

<iframe src="./assets/partial-trailing.html" title="分批止盈与剩余仓位管理" loading="lazy" style="width:100%;height:470px;border:0;display:block;"></iframe>

假设：

```text
入场：7504
止损：7499
第一目标：7509
```

到达第一目标后，可以先减仓一部分，剩余仓位继续观察。不要因为刚刚出现一点浮盈，就把全部仓位平掉。

只有完成下面的结构，才移动剩余仓位止损：

```text
产生位移
→ 形成新的更高低点
→ 再次启动
→ 止损上移到新结构外侧
```

如果接近供给区后连续冲不上去，或者出现实体跌回突破边界，应继续减仓甚至全部退出。

## 三、加仓只发生在盈利的新结构之后

<iframe src="./assets/add-vs-average-down.html" title="顺势加仓与亏损补仓对比" loading="lazy" style="width:100%;height:500px;border:0;display:block;"></iframe>

可以加仓的条件：

- 原仓位已经产生有利位移；
- 形成新的更高低点或更低高点；
- 新止损可以上移；
- 加仓后总风险不超过原定 `1R`；
- 前方仍有足够目标空间。

如果价格下跌后只是变得更便宜，但结构没有重新证明正确，就不能补仓。

### 风险例子

```text
本笔最大风险：$100
原仓位风险：$60
形成新结构并上移止损后，原仓位剩余风险：$20
新仓位风险：$50
总风险：$70，仍在预算内
```

如果原仓位仍承担 `$60` 风险，又新增 `$60` 风险，总风险就是 `$120`，即使方向很强也不能直接加仓。

## 四、目标节点与完整生命周期

<iframe src="./assets/trade-lifecycle.html" title="一笔交易的完整生命周期" loading="lazy" style="width:100%;height:520px;border:0;display:block;"></iframe>

一笔多头交易可以按下面的顺序管理：

```text
需求区观察
→ ①突破
→ ②回踩守住
→ ③确认建仓
→ 新结构加仓
→ 第一目标分批止盈
→ 剩余仓位跟随
→ 结构失效退出
```

目标区负责兑现利润，结构负责保护剩余仓位。GEX、资金流和节点可以辅助寻找目标，但不能替代价格结构。

对于0DTE，目标区附近更应该主动兑现一部分，因为时间价值、Gamma和买卖价差会让价格变化更快。

## 五、如何复盘一笔交易

复盘不能只写“今天赚了”或“今天亏了”，必须把计划、实际执行和结果分开。

### 4. 方向正确，也可能执行错误

例如：

```text
计划入场：7504
计划止损：7499
计划目标：7512
```

但实际追到 `7507` 才入场，导致剩余目标变小、盈亏比恶化。这种交易的结论应该是：

```text
方向判断正确，执行价格错误。
改进点：不要远离③追价。
```

不应该因为一次追价，就否定整个方向框架。

## 本章总结

```text
价格产生位移 → 持有
价格不动 → 降低风险
达到第一目标 → 分批兑现
形成新结构 → 才能移动止损或加仓
接近供给区 → 继续减仓
结构失效 → 退出
交易结束 → 对照计划复盘
```
